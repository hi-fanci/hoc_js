Arrow function trong Javascript

Arrow function là một dạng viết gọn của funtion.

Khai báo

Để khai báo arrow function trong Javascript, bạn dùng từ khoá => để định nghĩa.

  • let function_name = () => {
    • // code, nội dung của function
  • };

Cách sử dụng tương tự hàm đã được khai báo.

  • let say_hello = () => {
    • console.log('hello world');
  • }
  • say_hello(); // hello world
