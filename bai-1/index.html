Biến trong Javascript

Biến là một tên đại diện cho một giá trị và giá trị này có thể thay đổi.

Ví dụ, _x là một biến chỉ thời gian bao gồm giờ và phút, giá trị cuả _x hiện tại là , thì 10 phút nữa, giá trị của _x sẽ là

Cách khai báo biến

Để khai báo biến trong JavaScript, bạn dùng từ khóa let (ngoài ra còn có từ khóa var, nhưng hiện tại thì rất ít người dùng).

  • let variable_name;

Ví dụ, bạn muốn khai báo một biến tên là "my_name".

  • let my_name;

Việc khởi tạo biến đã thành công, bây giờ, bạn có thể gán giá trị cho biến "my_name", bằng cách dùng toán tử gán ( = ).

  • let my_name;
  • my_name = "John";
  • console.log( my_name ); // "John"

Để ngắn gọn hơn, bạn có thể khai báo và gán giá trị cùng lúc.

  • let my_name = "John";
  • console.log( my_name ); // "John"

Tương tư, bạn củng có thể khai báo nhiều biến và giá trị của nó cùng lúc thông qua dấu ( , ).

  • let my_name = "John",
    my_age = 20,
    my_gender = "man";
  • console.log( my_name ); // "John"
    console.log( my_age ); // 20
    console.log( my_gender ); // "man"

Quy tắc đặt tên của biến

Bắt đầu bằng chữ cái ( a-zA-Z ), kí tự ( _ ), kí tự ( $ ).
Không có các kí tự đặc biệt ( !@#%^&*()-=.... ).
Không bắt đầu bằng các số ( 0-9 ).
Không sử dụng từ khoá có sẳn của JavaScript.
break, case, catch, class, const, continue, debugger, default, delete, do, else, enum, export, extends, false, finally, for, function, if, implements, import, in, instanceof, interface, let, new, null, package, private, protected, public, return, static, super, switch, this, throw, true, try, typeof, var, void, while, with, yield

Tên biến trong JavaScript có phân biệt chữ thường và CHỮ HOA.

  • let my_name = "John",
    MY_NAME = "Anna";
  • console.log( my_name ); // "John"
    console.log( MY_NAME ); // "Anna"

Cách thay đổi giá trị của biến

Để thay đổi giá trị của biến, bạn chỉ cần gán giá trị mới cho nó.

  • let my_name = "John";

  • // thay đổi giá trị của my_name my_name = "Anna";
  • console.log( my_name ); // "Anna"

Bạn củng có thể thay đổi giá trị của biến này bằng một biến khác.

  • let my_name = "John",
    your_name = "Anna";

  • // thay đổi giá trị của my_name
    my_name = your_name;
  • console.log( my_name ); // "Anna"

Một số lỗi của biến

Trong một vùng phạm vi thực thế nhất định ( scope block ), thì việc khai báo một tên biến nhiều lần sẽ gây ra lỗi.

  • let my_name = "John";
  • let my_name = "Anna";
  • // Uncaught SyntaxError: Identifier 'my_name' has already been declared.

Hằng trong Javascript

Hằng là một tên đại diện cho một giá trị và giá trị này không thay đổi.

Ví dụ, số PI trong toán học có giá trị không đổi là 3.14, do vậy PI được gọi là một hằng số.

Cách khai báo hằng

Để khai báo hằng trong JavaScript, bạn dùng từ khóa const, và hằng phải có giá trị đi kèm, nếu không sẽ lỗi.

  • const variable_name = "value";

Ví dụ, bạn muốn khai báo một hằng tên là "PI".

  • const PI = 3.14;
  • console.log( PI ); // 3.14

Bạn củng có thể khai báo nhiều hằng thông qua dấu ( , ).

  • const PI = 3.14,
    VN = "Việt Nam",
    EN = "English";
  • console.log( PI ); // 3.14
    console.log( VN ); // "Việt Nam"
    console.log( EN ); // "English"
