Function trong Javascript

function ( hàm ) là một chương trình giúp thực hiện một công việc cụ thể, có thể tái sử dụng trong nhiều trường hợp, dự án khác nhau.

Khai báo

Để khai báo hàm trong Javascript, bạn dùng từ khoá function

  • function function_name () {
    • // code, nội dung của function
  • };

Để sử dụng hàm đã được khai báo, bạn chỉ việc gọi hàm theo tên.

  • function say_hello () {
    • console.log('hello world');
  • }
  • say_hello(); // hello world

Tham số của hàm

Tham số là đại diện cho 1 giá trị bên ngoài được truyền vào, một hàm có thể có 0 hoặc nhiều tham số khác nhau.
Tham số có thể được cài mặc định khi không có quá nhiều sự biến đổi ở một số trường hợp nhất định.

  • function say_something (message = "hehe") {
    • console.log(message);
  • }
  • say_something(); // hehe
  • say_something("hello"); // hello
  • say_something("hihi"); // hihi

Biến trong hàm

Biến trong hàm hay còn được biết là biến cục bộ.
Có nghĩa là biến được khai báo ở hàm nào thì chỉ có thể sử dụng ở trong hàm đó.

  • function say_something () {
    • let _msg = "hehe";
    • console.log(_msg); // hehe
  • }
  • console.log(_msg); // Uncaught ReferenceError: _msg is not defined

Biến ngoài hàm

Biến ngoài hàm hay còn được biết là biến toàn cục.
Có nghĩa là biến được khai báo ở ngoài hàm, được sử dụng, thay đổi giá trị nếu cần.

  • let _msg = "hehe";
  • function say_something () {
    • console.log(_msg); // hehe
    • _msg = "hihi";
  • }
  • console.log(_msg); // hihi

Return của hàm

Để trả về một kết quả, của một chương trình con, bạn có thể dùng từ khoá return.

Ví dụ, ta có một hàm tính tổng 2 số:

  • function sum (a, b) {
    • return a + b;
  • }
  • console.log( sum(1, 2) ); // 3
  • console.log( sum(10, 20) ); // 30

Ngoài ra, return có thể đặt bất cứ đâu trong hàm, chú ý phải đảm bảo tính logic.

  • function is_number_0 (n) {
    • if( n == 0 ) return "yes";
    • return "no";
  • }
  • console.log( is_number_0(10) ); // no
  • console.log( is_number_0(0) ); // yes
