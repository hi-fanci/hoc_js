Thêm style CSS vào html

Có 2 cách để thêm style css vào file html

  • Thêm bằng file
    • <link rel="stylesheet" href="[path]/css/[file name].css">
    • [path] là đường dẫn tới file css
    • Ở đầu các file phải có css rule @charset "UTF-8";
  • Thêm trực tiếp
    • style
      // code css here
      /style

    <style></style> thường được đặt trong thẻ <head></head>

Cách dùng Css

Có 2 cách dùng css phổ biến.

  • Viết css ở file hoặc trong thẻ <style>
    • .đối-tượng {
      thuộc-tính:
      giá-trị
      }
  • Viết css inline
    • <p style="thuộc-tính: giá-trị;"></p>

Selector ( bộ chọn ) của css

  • Chọn theo html tag
    • tag-name{}
      // ví dụ: html{}, body{}, p{}
  • Chọn theo id
    • #id{}
      // ví dụ: #wraper{}, #input{}, #btn{}
  • Chọn hết
    • *{}
      // ví dụ: *{}, my-block *{}, your-block *{}
  • Chọn theo class
    • .class-name{}
      // ví dụ: .toggler{}, .confirm{}, .submit{}
  • xem thêm ở đây
  • class có phân biệt chữ thường và CHỮ HOA.
mức độ ưu tiên mặc định của css là thuộc tính ở dưới sẽ ghi đè thuộc tính của dòng trên.

:hover{}

:hover sẽ ghi đè giá trị của các thuộc tính, thường dùng cho các trường hợp làm hiệu ứng rê chuột, ví dụ:

  • .box {
    background-color:
    #009966;
    }
  • .box:hover {
    background-color:
    #06b6d4;
    }
hover me !

Đừng quên thêm thuộc tính "cursor: pointer", "transition".

Transition

  • transition: property duration timing-function;
  • property: thuộc tính cần thay đổi, thường dùng all hoặc để trống, xem thêm.
  • duration: thời gian diễn ra sự thay đổi, có thể là giây, mili giây, 1000ms = 1s.
  • timing-function: là cách mà sự thay đổi diễn ra, xem thêm.
  • .box {
    background-color:
    #009966;
    transition:
    450ms ease-out;
    cursor:
    pointer;
    }
  • .box:hover {
    background-color:
    #06b6d4;
    }
hover me !
