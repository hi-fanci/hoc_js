Các toán tử trong Javascript

Toán tư trong JS củng tương tự các toán tử cơ bản của toán học, như dấu +, -, *, /

Các kiểu toán tử thông dụng

Toán tử số học

Toán tử số học dùng để thực hiện các phép tính toán trong số học. bao gồm:

Toán tử cộng +
Toán tử trừ -
Toán tử nhân *
Toán tử luỹ thừa **
Toán tử chia /
Toán tử chia lấy số dư%
  • console.log(3 + 2); // 5
  • console.log(3 - 2); // 1
  • console.log(3 * 2); // 6
  • console.log(3 ** 2); // 9 ( 3 ^ 2)
  • console.log(3 / 2); // 1.5
  • console.log(3 % 2); // 1 ( số dư )

Toán tử nối chuỗi

Khi một trong hai toán hạng của toán tử của + là kiểu string thì toán hạng còn lại sẽ bị chuyển thành kiểu string.

  • console.log("hello" + "world"); // hello world
  • console.log("1" + 2); // 12

Toán tử tăng/giảm

Đây là cú pháp ngắn gọn cho việc tăng/giảm giá trị cho 1 biến

  • let x = 10; // x = 10
  • x ++; // x = 11
  • x += 4; // x = 15
  • x --; // x = 14
  • x -= 4; // x = 10

Toán tử điều kiện rẽ nhánh

Toán tử này dùng cho câu lệnh If- Else cách ngắn gọn

if true : thực hiện điều kiện đúng : điều kiện sai

  • console.log(( 1 > 10 ) ? "yes" : "no"); // no
  • console.log(( 10 > 1 ) ? "yes" : "no"); // yes
