Kiểu dữ liệu là gì?

Kiểu dữ liệu là cách phân loại dữ liệu cho trình biên dịch hiểu về kiểu của biến đang dùng.

Các kiểu dữ liệu thông dụng

Kiểu dữ liệu boolean (kiểu logic)

Boolean là kiểu dữ liệu logic chỉ bao gồm hai giá trị là true và false.

  • let the_week_has_10_day = false;
  • let the_week_has_7_day = true;

Kiểu dữ liệu null

Kiểu dữ liệu null có nghĩa là chưa có giá trị cụ thể, tạm thời set là null.

  • let the_week_is_coming = null;

Kiểu dữ liệu undefined

Kiểu dữ liệu undefined có nghĩa là giá trị chưa được gán, có thể hiểu là giá trị mặc định của biến sau khi tạo và không gán giá trị cụ thể.

  • let name; //undefined

Kiểu dữ liệu number

Kiểu dữ liệu number là kiểu dữ liệu dạng số, tương tự trong toán học, có 3 loại thông dụng là số nguyên, số thực và NaN ( không phải số ).

  • let n = 1 ; // số nguyên dương 1
    n = -1 ; // số nguyên âm 1
    n = 1.1 ; // số thực dương 1.1
    n = -1.1 ; // số thực âm 1.1

NaN: là viết tắt của Not a Number, được sử dụng để đại diện cho những trường hợp tính toán sai hoặc kết quả của một phép tính không xác định.

  • lấy số 0 chia cho số 0
    console.log(0 / 0); // NaN
  • lấy 1 chuỗi kí tự chia cho 1 số
    console.log(:"hi umi" / 100); // NaN

Kiểu dữ liệu string

Kiểu dữ liệu string để biểu diễn chữ, văn bản, đoạn văn bản,...
Có ba cách để biểu diễn string trong JavaScript: "", '', ``.

  • let name = "hi umi"; //dùng dấu "
  • name = 'hi umi'; //dùng dấu '
  • name = `hi umi`; //dùng dấu `

Đặc biệt là dấu `` có thể thêm biến, hằng, biểu thức bên trong, bằng cách dùng ${}.

  • let name = "hi umi";
    console.log(`my name is ${ name }`); //my name is hi umi
  • let age = 100;
    console.log(`your age is ${ age + 100 }`); //your age is 200
  • let is_true = true;
    console.log(`is it true? ${ is_true ? 'yes' : 'no' }`); //is it true? yes
    is_true = false;
    console.log(`is it true? ${ is_true ? 'yes' : 'no' }`); //is it true? no

Kiểu dữ liệu object

Object là kiểu dữ liệu tham chiếu. Có thể hiểu object là một tập hợp gồm các cặp key - value (khóa - giá trị).

  • const person = {
    name:
    "hi umi",
    age:
    100
    }

Bạn có thể lấy giá trị của object bằng 2 cách, dùng . hoặc [].

  • console.log(person.name); // "hi umi"
    console.log( person[ name ]); // "hi umi"

Cách xác định kiểu dữ liệu của biến

Để xác định kiểu dữ liệu hiện tại của một biến, bạn có thể dùng typeof với hai loại cú pháp:

Dạng toán tử typeof x.
  • let name;
    console.log(typeof name); //undefined
  • name = "hi umi";
    console.log(typeof name); //string
  • name = true;
    console.log(typeof name); //boolean
  • name = 100;
    console.log(typeof name); //number
  • name = { x: 0 };
    console.log(typeof name); //object
Dạng hàm typeof(x).
  • let name;
    console.log(typeof (name)); //undefined
  • name = "hi umi";
    console.log(typeof (name)); //string
  • name = true;
    console.log(typeof (name)); //boolean
  • name = 100;
    console.log(typeof (name)); //number
  • name = { x: 0 };
    console.log(typeof (name)); //object
