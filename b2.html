Hằng là gì?

Tương tự biến, hằng củng là "tên tượng trưng" dùng để đại diện cho một giá trị không đổi.

Ví dụ: số PI được gọi là một hằng vì số PI có giá trị không đổi là 3.14159.

Cách khai báo hằng

Để khai báo hằng trong JavaScript, bạn dùng từ khóa const.

  • const tên hằng = giá trị;

Ví dụ: khai báo số PI.

  • const PI = 3.14159;

Bạn có thể khai báo hằng và gán giá trị cho chúng trên cùng một lần khai báo:

  • const PI = 3.14159, PC = "Personal Computer", GG = "Good Game";

Hoặc là nguyên bản:

  • const PI = 3.14159;
  • const PC = "Personal Computer";
  • const GG = "Good Game";

Thay đổi giá trị của hằng

Hằng số là một giá trị không đổi.

Một số lỗi thường gặp

Khai báo hằng không có giá trị

Ví dụ:

  • const name; //Uncaught SyntaxError: Missing initializer in const declaration.

Bạn sẽ bị lỗi: Uncaught SyntaxError: Missing initializer in const declaration.

Khai báo lại hằng đã có

Ví dụ:

  • const name = "hihi";
  • const name = "hello world"; //Uncaught SyntaxError: Identifier 'a' has already been declared.

Bạn sẽ bị lỗi: Uncaught SyntaxError: Identifier 'a' has already been declared.

Thay đổi giá trị của hằng.

Ví dụ:

  • const name = "hihi";
    name = "hello world"; //Uncaught TypeError: Assignment to constant variable.

Hằng số không thể thay đổi giá trị. Nếu bạn cố gắng thay đổi giá trị thì có lỗi như sau: Uncaught TypeError: Assignment to constant variable.
