Thêm css style vào html

  • Thêm bằng file:

    Trước hết, mình cần 1 file css, và đường dẫn(path) từ file html đến file css đó.
    Ở đầu các file css nên có css rule: @charset "UTF-8";

    • <link rel="stylesheet" href="[path]/[file name].css">
  • Thêm trực tiếp:

    Thẻ style được đặt bên trong thẻ head

    • style
      // code css here
      /style

Cách viết css

  • file css
    hoặc thẻ style    :

    Viết ở file css, hay trong thẻ style đều giống nhau.
    Chỉ cần có 1 bộ chọn phần tử, sau đó ghi các thuộc tính cần css trong dấu {}

    • bộ-chọn{
      thuộc-tính:
      giá-trị
      }
  • css inline:

    Cách này khó sử dụng hơn, độ hiệu quả củng không mấy cao, nhưng được rất nhiều người tin dùng.

    • p style="thuộc-tính: giá-trị;"/p

Selector(bộ chọn) của css

  • Chọn html tag:

    Các tag name là các thẻ trong html
    Ví dụ: html, body, p

    • tag-name{}
  • Chọn id:

    id dùng để định nghĩa 1 phần tử duy nhất trong trang, và không được trùng lặp.
    Phải có dấu # ở phía trước để phân biệt.
    Ví dụ: #html, #wraper, #input-submit

    • #id-name{}
  • Chọn class:

    class dùng để định nghĩa 1 hoặc nhiều phần tử có thuộc tính giống nhau, gần giống nhau trong trang, thường được dùng lại nhiều lần.
    Phải có dấu . ở phía trước để phân biệt.
    Ví dụ: .block, .btn, .text

    • .class-name{}
  • Chọn hết:

    Lấy tất cả các phần tử bên trong(phần tử con) của phần tử được chọn.

    • bộ-chọn *{}
mức độ ưu tiên mặc định của css là thuộc tính ở dưới sẽ ghi đè thuộc tính của dòng trên.
  • span {
    color:
    red; // thuộc tính này sẽ bị loại bỏ
    color:
    yellow; // do bị ghi đè ở đây
    }
    • inline style
    • id
    • class, :hover
    • thuộc tính
    • thẻ html ::after, ::before

:hover

:hover sẽ ghi đè giá trị của các thuộc tính hiện tại, sử dụng cho hiệu ứng rê chuột, ví dụ:

  • .box {
    background-color:
    #009966;
    }
  • .box:hover {
    background-color:
    #06b6d4;
    }
hover me !

Đừng quên thêm thuộc tính "cursor: pointer" và "transition".

Transition

  • transition: property duration timing-function;
  • property: thuộc tính cần thay đổi, thường dùng all hoặc để trống, xem thêm.
  • duration: thời gian diễn ra sự thay đổi, có thể là giây, mili giây, 1000ms = 1s.
  • timing-function: là cách mà sự thay đổi diễn ra, xem thêm.
  • .box {
    background-color:
    #009966;
    transition:
    450ms ease-out;
    cursor:
    pointer;
    }
  • .box:hover {
    background-color:
    #06b6d4;
    }
hover me !

@media rule

Quy tắc @media được dùng để áp dụng các styles khác nhau cho các màn hình tương ứng.

  • @media only screen and (min-width: xxx)

    xxx sẽ xác định độ rộng màn hình nhỏ nhất được áp dụng css.

    • div{
      color:
      black;
      }
      @media only screen and (min-width: 1024px){
      div{
      color:
      red;
      }
      }
    Dòng này sẽ có màu "red(đỏ)" khi độ rộng màn hình lớn hơn 1024px, và ngược lại thì màu "black(đen)".
  • @media only screen and (max-width: xxx)

    xxx sẽ xác định độ rộng màn hình lớn nhất được áp dụng css.

    • div{
      color:
      black;
      }
      @media only screen and (max-width: 1024px){
      div{
      color:
      red;
      }
      }
    Dòng này sẽ có màu "red(đỏ)" khi độ rộng màn hình nhỏ hơn 1024px, và ngược lại thì màu "black(đen)".
  • @media only screen and (min-width: xxx) and (max-width: yyy)

    Tương tự, xxx, yyy sẽ lần lượt là giới hạn nhỏ nhất, lớn nhất được áp dụng css.

    • div{
      color:
      black;
      }
      @media only screen and (min-width: 768px) and (max-width: 1024px){
      div{
      color:
      red;
      }
      }
    Dòng này sẽ có màu "red(đỏ)" khi độ rộng màn hình trong khoảng từ 768px đến 1024px, và ngược lại thì màu "black(đen)".

để thuận tiện cho việc thêm, sửa, xoá thuộc tính sau này, tui khuyến khích dùng @media ngay sau khối khai báo của selector
