Object

Object trong JavaScript là một khái niệm trừu tượng dùng để biểu diễn một vật thể ( cụ thể ). Trong đó bao gồm các thuộc tính , phương thức dùng để miêu tả đặc điểm, tính chất của đối tượng.

  • let person = {
    name:
    'John',
    age:
    100
    };

Khởi tạo

  • { }
    • // Tạo object rỗng
      let person = { };

      // Tạo object có dữ liệu
      let person = {
      name:
      'John'
      };

    Cách này phổ biến vì ngắn gọn, dễ sử dụng.

Đặt Tên thuộc tính (key)

  • let person = {
    name:
    'John', // true
    _name:
    'John', // true
    $name:
    'John', // true
    My_name':
    'John', // true
    '123 !@# my name':
    'John', // true

    2 my name:
    'John', // wrong
    ! my name:
    'John', // wrong
    };
  • Tên thuộc tính (key) của object chỉ được bắt đầu bằng a-z, A-Z, _ và $, nếu ở dạng chuỗi thì không có vấn đề gì.
  • Tên thuộc tính (key) không được trùng với các từ khoá có sẳn của Javascript, tương tự cách đặt tên biến

Truy cập vào thuộc tính

  • Toán tử .
    • let person = {
      name:
      'John',
      };

    • console.log( person.name );
      // "John"
    • Toán tử . chỉ dùng được với các thuộc tính không là chuỗi.
  • [ ]
    • let person = {
      name:
      'John',
      };

    • console.log( person[ 'name' ] );
      // "John"
  • Biến là tên của thuộc tính
    • let person = {
      name:
      'John',
      };
      let prop = 'name';

    • console.log( person[ prop ] );
      // "John"

Thêm, thay đổi thuộc tính

  • Toán tử .
    • let person = {
      name:
      'John',
      };
    • person.age = 100;

    • console.log( person );
      // { "name": "John", "age": 100 }
    • Toán tử . chỉ dùng được với các thuộc tính không là chuỗi.
  • [ ]
    • let person = {
      name:
      'John',
      };
    • person[ 'age' ] = 100;

    • console.log( person );
      // { "name": "John", "age": 100 }
  • Biến là tên của thuộc tính
    • let person = {
      name:
      'John',
      };
      let prop = 'age';
    • person[ prop ] = 100;

    • console.log( person );
      // { "name": "John", "age": 100 }

Phương thức của Object

  • let person = {
    name:
    'John'
    sayHello:
    function(){
    • console.log( 'hello world );
    }
    };

  • console.log( person.sayHello() );
    // "hello world"
  • function là một thuộc tính của object thì nó được xem như là một phương thức của object.

Clone object

  • spread { ...object }
    • let person = {
      name:
      'John',
      };
      let person_clone = { ...person };

    • console.log( person_clone );
      // { name: 'John'}
    • Phương thức này chỉ dùng cho object chứa các kiểu dữ liệu nguyên thuỷ, xem thêm ở đây.
  • JSON.stringify( ) và JSON.parse( )
    • let person = {
      name:
      'John',
      hobby:
      {
      colors:
      'red',
      number:
      100
      }
      };
      let person_clone = JSON.parse( JSON.stringify( person ) );

    • console.log( person_clone );
      // { name: 'John', hobby: { color: 'red', number: 100}}
  • Phương thức này không thể clone được phương thức của object.
