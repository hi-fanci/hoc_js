Color

  • p {
    color:
    #06b6d4;
    }

This is p with color: #06b6d4

Ngoài ra, chúng ta có màu dạng rgba(red, green, blue, alpha), trong đó RGB là mã màu, alpha là độ trong suốt, có giá trị từ 0-1.

  • p {
    color:
    rgba(6, 182, 212, 0.5);
    }

This is p with color: rgba(6, 182, 212, 0.5)

Background

background có rất nhiều loại, bao gồm background-color, background-image, background-clip

  • background-color
    • .box{
      background-color:
      giá trị;
      }
    background-color: #06b6d4

    background-color chỉ nhận các màu đơn sắc, như tên màu ( red, black ... ), hex, rgb, rgba ....
    Đặc biệt currentColor sẽ cho màu tương ứng với màu chữ hiện tại.

  • background-image: gradient
    • .box{
      background-image:
      giá trị;
      }
    linear-gradient
    (45deg, #009966, #06b6d4);
    linear-gradient
    (45deg, #f6339a, #06b6d4);

    Xem thêm các thuộc tính khác của gradient.

  • background-image: url
    • .box{
      background-image:
      url(đường dẫn);
      }

    Xem thêm các thuộc tính khác của background.

    background-url thường dùng cho các layout dùng chung, như là icon, MV của các trang con, hoặc là các trường hợp dùng lớp giả như ::after, ::before.

  • background-clip
    • .box{
      background-clip:
      text;
      -webkit-background-clip:
      text;
      }
    this is text
    this is text

    Xem thêm các thuộc tính khác của background-clip.

    background-clip thường dùng để thay đổi màu, hình nền cho chữ, văn bản.
