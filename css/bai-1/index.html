Thêm style CSS vào html

Có 2 cách để thêm style css vào file html

  • Thêm bằng file
    • <link rel="stylesheet" href="[path]/css/[file name].css">
    • [path] là đường dẫn tới file css
    • Ở đầu các file phải có css rule @charset "UTF-8";
  • Thêm trực tiếp
    • style
      // code css here
      /style

    <style></style> thường được đặt trong thẻ <head></head>

Cách dùng Css

Có 2 cách dùng css phổ biến.

  • Viết css ở file hoặc trong thẻ <style>
    • .đối-tượng {
      thuộc-tính:
      giá-trị
      }
  • Viết css inline
    • <p style="thuộc-tính: giá-trị;">Lorem, ipsum.</p>

Selector ( bộ chọn ) của css

  • Chọn theo html tag
    • tag-name{}
      // ví dụ: html{}, body{}, p{}
  • Chọn theo id
    • #id{}
      // ví dụ: #wraper{}, #input{}, #btn{}
  • Chọn hết
    • *{}
      // ví dụ: *{}, my-block *{}, your-block *{}
  • Chọn theo class
    • .class-name{}
      // ví dụ: .toggler{}, .confirm{}, .submit{}
  • xem thêm ở đây
  • Tên class có phân biệt chữ thường và CHỮ HOA.
mức độ ưu tiên mặc định của css là thuộc tính ở dưới sẽ ghi đè thuộc tính của dòng trên.
  • span {
    color:
    red; // thuộc tính này sẽ bị loại bỏ
    color:
    yellow; // do bị ghi đè ở đây
    }
Thứ tự sẽ được xếp theo độ ưu tiên dưới đây.

1. Inline style <p style="color: #333;">

2. Id #element

3. Class & :hover #element

3. Thuộc tính [type="text"]

3. Tag & ::after, ::before h1, ::after, ::before

Thay đổi giá trị thuộc tính bằng :hover{}

Chỉ cần thêm thuộc tính cần thay đổi vào trong :hover{}.
Ví dụ mình có một thẻ span đang được style là màu đỏ ( red ), và hover chuyển thành màu vàng (yellow).

  • span {
    color:
    red;
    }
  • span:hover {
    color:
    yellow;
    }
Đây là thẻ p màu đỏ

Transition

Củng là ví dụ như trên, nhưng có thêm transition.

  • transition: property duration timing-function;
  • property: là thuộc tính cần thay đổi, xem thêm.
  • duration: là thời gian diễn ra sự thay đổi thuộc tính, có thể là giây, mili giây, 1000ms = 1s.
  • timing-function: là cách mà sự thay đổi diễn ra, xem thêm.
  • span {
    color:
    red;
    }
  • span:hover {
    color:
    yellow;
    transition:
    color 2s ease;
    }
Đây là thẻ p màu đỏ

Cách khai báo hằng

Để khai báo hằng trong JavaScript, bạn dùng từ khóa const, và hằng phải có giá trị đi kèm, nếu không sẽ lỗi.

  • const variable_name = "value";

Ví dụ, bạn muốn khai báo một hằng tên là "PI".

  • const PI = 3.14;
  • console.log( PI ); // 3.14

Bạn củng có thể khai báo nhiều hằng thông qua dấu ( , ).

  • const PI = 3.14,
    VN = "Việt Nam",
    EN = "English";

  • console.log( PI ); // 3.14
    console.log( VN ); // "Việt Nam"
    console.log( EN ); // "English"
