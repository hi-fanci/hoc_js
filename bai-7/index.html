localStorage

localStorage là một API cho phép lưu trữ dữ liệu dưới dạng chuỗi là một JSON, gồm cặp key-value (khóa-giá trị) trong trình duyệt của người dùng.
Tối đa là 5mb cho mỗi tên miền, tuỳ thuộc vào trình duyệt.
Dữ liệu được lưu trữ trong localStorage sẽ tồn tại cho đến khi người dùng xóa nó bằng cách thủ công hoặc qua mã JavaScript.
Điều này có nghĩa là dữ liệu sẽ không bị mất khi tải lại trang hoặc đóng trình duyệt.

  • localStorage.setItem( 'key', 'value' );

Khởi tạo

  • Trực tiếp
    • localStorage.setItem( 'say', 'hello world' );

    Cách này phổ biến vì ngắn gọn, dễ sử dụng.

  • Dùng biến
    • let key = 'say';
      let val = 'hello world';
    • localStorage.setItem( key, val );

    Cách này tối ưu hơn và dễ bảo trì.

Lấy giá trị theo từ khoá từ localStorage

  • let say = localStorage.getItem('say');
  • console.log( say ); // "hello world"
  • let key = 'say';
  • let say = localStorage.getItem( key );
  • console.log( say ); // "hello world"

Xoá giá trị theo từ khoá từ localStorage

  • localStorage.removeItem('say');
  • let say = localStorage.getItem('say');
  • console.log( say ); // null
  • let key = 'say';
  • localStorage.removeItem( key );

  • let say = localStorage.getItem( key );
  • console.log( say ); // null

Xoá tất cả dữ liệu của trang trong localStorage

  • localStorage.clear();
  • let say = localStorage.getItem('say');
  • console.log( say ); // null
  • let key = 'say';
  • localStorage.clear();

  • let say = localStorage.getItem( key );
  • console.log( say ); // null
