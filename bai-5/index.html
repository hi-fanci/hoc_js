setTimeout

setTimeout là một hàm được thực hiện sau một khoảng thời gian nhất định, được tính bằng mili giây (ms), chỉ chạy 1 lần, nếu không được gọi lại.

  • setTimeout( function(){
    • // code here
    }, time);

Nếu time là 0, thì hàm sẽ không chạy liền, mà bị xếp vào cuối cùng so với các hành động cùng cấp.

  • setTimeout( function(){
    • console.log( 'hello world' );
    }, 0);
  • console.log( 'xin chao' );
  • console.log( 'woww' );

  • // "xin chao"
    // "woww"
    // "hello world"

Khởi tạo

  • Trực tiếp
    • setTimeout( function(){
      • console.log( 'hello world' );
      }, 100);

    • // sau 100ms (0.1s)
      "hello world"

    Cách này phổ biến vì ngắn gọn, dễ sử dụng.

  • Dùng function
    • const sayHello = () => {
      • console.log( 'hello world' );
      };
    • setTimeout( function(){
      • sayHello();
      }, 100);

    • // sau 100ms (0.1s)
      "hello world"

    Cách này tối ưu hơn và dễ bảo trì.

Huỷ hàm với clearTimeout

Để có thẻ huỷ một hàm setTimeout thì mình cần phải có một định danh của riêng nó, tương tự như một id.

  • let say = setTimeout( function(){
    • console.log( 'hello world' );
    }, 1000);

Hiện tại mình đã gán setTimeout cho biến say, và sau 1000ms ( 1s ) thì nội dung của hàm sẽ được hoạt động.

Bây giờ, mình sẽ huỷ bỏ bằng cách dùng clearTimeout.

  • clearTimeout( say );

Khi clearTimeout được gọi trước khi hết thời gian chờ của hàm setTimeout thì việc huỷ bỏ sẽ thành công.

Bài trước: Object - nâng cao.
Bài tiếp: Hàm setInterval - cơ bản.
