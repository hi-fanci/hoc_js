font-family

font-family còn gọi là kiểu chữ, giúp văn bản, kí tự thay đổi cách hiện thị theo cài đặt.

font-family bao gồm 2 giá trị: "tên kiểu chữ", nhóm của kiểu chữ

  • p {
    font-family:
    "Open Sans", sans-serif;
    }

font-family: "Open Sans"

font-family: "Inconsolata"

font-weight

font-weight còn gọi là độ đậm, giúp văn bản, kí tự thay đổi độ đậm nhạt theo cài đặt và theo font-family.

font-weight có các giá trị phổ biến như là: thin-100, extraLight-200, light-300, regular-400, medium-500, semiBold-600, bold-700, extraBold-800, black-900

  • p {
    font-weight:
    500;
    }

font-weight: 300

font-weight: 500

font-weight: 700

font-size

font-size còn gọi là cở chữ, giúp văn bản, kí tự thay đổi cở chữ theo cài đặt.

font-size có các giá trị phổ biến như là: px, em, rem, vw

  • p {
    font-size:
    10px;
    }

font-size: 20px

font-size: 2em

font-size: 2rem

font-size: 2vw

font-style

font-style còn gọi là kiểu dáng, giúp văn bản, kí tự tạo chữ nghiêng theo cài đặt.

font-style có các giá trị phổ biến như là: normal(bình thường), italic(in nghiêng), oblique(hơi nghiêng).

  • p {
    font-style:
    italic;
    }

font-style: normal

font-style: oblique

font-style: italic

line-height

line-height còn gọi là chiều cao dòng, giúp văn bản, kí tự tạo ra khoảng cách các dòng theo cài đặt.

line-height có các giá trị phổ biến như là: số, px, em, rem.

  • p {
    line-height:
    1.5; // 1.5 = 150% font-size hiện tại
    }

line-height: 1.5

line-height: 20px

line-height: 5em

letter-spacing

letter-spacing còn gọi là khoảng cách kí tự, giúp văn bản, kí tự tạo ra khoảng cách với nhau theo cài đặt.

letter-spacing có các giá trị phổ biến như là: %, px, em, rem.

  • p {
    letter-spacing:
    0.5em;
    }

letter-spacing: 0.5em

letter-spacing: 5px

letter-spacing: 1rem

text-transform

text-transform còn gọi là biến đổi kiểu chữ, giúp văn bản, kí tự có thể viết hoa, thường theo cài đặt.

text-transform có các giá trị phổ biến như là: uppercase(viết hoa), lowercase(viết thường), capitalize(viết hoa chữ đầu).

  • p {
    text-transform:
    uppercase;
    }

text-transform: uppercase

text-transform: lowercase

text-transform: capitalize
