Object

Object trong JavaScript là một khái niệm trừu tượng dùng để biểu diễn một vật thể ( cụ thể ). Trong đó bao gồm các thuộc tính , phương thức dùng để miêu tả đặc điểm, tính chất của đối tượng.

  • let my_self = {
    name
    "John"
    age
    100
    };

Khai báo

Để khai báo một object, mình có hai cách như sau.

  • // khai báo với object rỗng, không chứa thuộc tính
    let object_01 = {};

  • // dùng hàm khởi tạo của object
    let object_02 = new Object();

Khởi tạo

  • { }
    • let obj = { };
      let xxx = {
      name
      "John"
      };

    Cách này được phổ biến do ngắn gọn, sử dụng.

Cách truy cập vào thuộc tính Object

Mình có một Object tên là my_self, bao gồm các thuộc tính là name( tên ), age( tuổi ), girl friend( ngừi iu ).

  • let my_self = {
    name
    'John'
    age
    100
    'girl friend'
    false
    };

Để truy cập vào Object thì mình có các cách như sau.

Sử dụng toán tử . kèm tên thuộc tính.
  • console.log( my_self.name ); // "John"
  • console.log( my_self.age ); // 100
  • console.log( my_self.'girl friend' ); // Uncaught SyntaxError: Unexpected string

Toán tử . chỉ dùng được với cả thuộc tính không chứa kí tự đặc biêt, trừ _ và $.

Sử dụng toán tử [] chứa tên thuộc tính.
  • console.log( my_self[ 'name' ] ); // "John"
  • console.log( my_self.[ 'age' ] ); // 100
  • console.log( my_self[ 'girl friend' ] ); // false

Nếu key là một chuỗi có dấu cách, mình cần phải có dấu nháy ', ", ` để hiểu đó là một chuỗi.

Sử dụng biến là tên của thuộc tính.
  • let prop_name = 'name';
    let prop_age = 'age';
    let prop_girl_friend = 'girl friend';
  • console.log( my_self[ prop_name ] ); // "John"
  • console.log( my_self[ prop_age ] ); // 100
  • console.log( my_self[ prop_girl_friend ] ); // false

Cách sử dụng Object

Cách đặt tên thuộc tính.

Tên thuộc tính của object chỉ được bắt đầu bằng a-zA-Z, _ và $, nếu ở dạng chuỗi thì không vấn đề gì.

  • let my_self = {
    name
    "John"
    _age
    100
    $salary
    '1000$'
    '!@#$%^&*() is good'
    true
    0or1
    '1000$' // Uncaught SyntaxError: Invalid or unexpected token
    !@#special
    '1000$' // Uncaught SyntaxError: Unexpected token '!'
    };
  • console.log( my_self[ 'name' ] ); // "John"
  • console.log( my_self[ '_age' ] ); // 100
  • console.log( my_self[ '$salary' ] ); // '1000$'
  • console.log( my_self[ '!@#$%^&*() is good' ] ); // true

Hai thuộc tính 0or1 và !@#special sẽ bị lỗi như trên, chỉ cần ném nó vào một string là xong.

Ngoài ra, tên thuộc tính không được trùng với từ khoá có sẳn của Javascript, tương tự cách đặt tên biến.

number

Number là kiểu dữ liệu dạng số (tương tự trong toán học). Number trong JavaScript không có cú pháp gì đặc biệt. Mình chỉ cần viết số ra, có hai loại số là: int ( số nguyên ) và float ( số thực ).

  • let x = 1, // số nguyên dương
    x = 1.11, // số thực dương
    x = -1, // số nguyên âm
    x = -1.11, // số thực âm

string

String là kiểu dữ liệu dùng để biểu diễn chữ, văn bản, đoạn văn bản,...
Có ba cách để biểu diễn string trong JavaScript.

Dùng dấu nháy đơn '
Dùng dấu nháy đôi "
Dùng dấu backtick `
  • let my_name = "John";
  • my_name = 'Anna';
  • my_name = `Anna`;

Đặc biệt, đối với dấu `, mình có thể sử dụng biến, hằng hoặc thậm chí viết một biểu thức trong đó, với cú pháp ${...}, ví dụ:

  • // Sử dụng biến
    let my_name = "John";
  • console.log( `My name is ${ my_name }` ); // "My name is John"

  • // Sử dụng với biểu thức
    console.log( `1 + 1 = ${ 1 + 1 }` ); // "1 + 1 = 2"

object

Object là kiểu dữ liệu tham chiếu, là một tập hợp gồm các cặp key - value ( khóa - giá trị ), được ngăn cách bởi dấu ,.
Trong đó, key có thể là string, value có thể là bất kì kiểu dữ liệu nào.

  • let my_self = {
    name
    "John"
    age
    100
    };

  • console.log( my_self.name ); // "John"
    console.log( my_self.age ); // 100
