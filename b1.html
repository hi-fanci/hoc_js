Biến là gì?

Biến là "tên tượng trưng" dùng để đại diện cho một giá trị và giá trị có thể thay đổi.

Giả sử x là số phút hiện tại.
Giá trị hiện tại là 5, thì 5 phút nữa, giá trị x sẽ thay đổi là 10.

Cách khai báo biến

Để khai báo biến trong JavaScript, bạn dùng từ khóa let (từ khóa var vẫn dùng được, nhưng rất ít người dùng).

Cú pháp khai báo biến như sau:
  • let tên biến;

Ví dụ: khai báo một biến tên là name

  • let name;

biến name đã được khởi tạo với giá trị mặc định là null, bạn có thể gán giá trị tuỳ ý thông qua toán tử =

Ví dụ:

  • name = "hi umi";

Bạn có thể khai báo biến và gán giá trị cho biến trên cùng lần khai báo:

  • let blabla, name = "hi umi", age = "100", company = "J line";

Hoặc là nguyên bản:

  • let blabla;
  • let name = "hi umi";
  • let age = "100";
  • let company = "J line";

Thay đổi giá trị của biến

Để thay đổi giá trị của biến, bạn chỉ cần gán giá trị mới cho nó là được.

Ví dụ:

  • let name = "hi umi";
    console.log(name); // hi umi
  • // Thay đổi giá trị
  • name = "hello world";
    console.log(name); // hello world

Một số lỗi thường gặp

Khai báo biến nhiều lần

Khi biến đã được khai báo, nếu bạn khai báo lại biến đó thì sẽ bị lỗi cú pháp.

Ví dụ:

  • let name = "hi umi";
  • let name = "hello world";

Bạn sẽ bị lỗi: Uncaught SyntaxError: Identifier 'name' has already been declared (biến "name" đã được định nghĩa rồi)

Gán giá trị cho biến mà không khai báo

  • age = 100;
    console.log(age); // 100

Thực tế thì bạn vẫn có thể gán giá trị cho biến khi chưa được khai báo.

Tuy nhiên ở chế độ "use strict" thì sẽ có lỗi: Uncaught ReferenceError: age is not defined (biến "age" chưa được định nghĩa).

Nhưng mình vẫn khuyên bạn nên khai báo rõ ràng cho các biến trước khi sử dụng.
