Kiểu dữ liệu trong Javascript

Kiểu dữ liệu là cách phân loại dữ liệu cho trình biên dịch hiểu về kiểu của biến đang dùng.

  • let xx = 10;
  • console.log( typeof x ); // number

  • xx = "I'm xx";
  • console.log( typeof x ); // string

Tổng quan các loại dữ liệu

JavaScript có 8 kiểu dữ liệu cơ bản, trong đó, có 7 kiểu dữ liệu nguyên thủy ( boolean, null, undefined, number, BigInt, string, symbol ) và 1 kiểu dạng tham chiếu ( object ).

Vì mình đang hướng dẫn cơ bản nên bạn chỉ cần chú tâm đến các kiểu sau ( boolean, null, undefined, number, string, object ).

bool ( kiểu logic )

Boolean là kiểu dữ liệu logic chỉ bao gồm hai giá trị là true ( đúng ) và false ( sai ), ví dụ.

  • let answer = ( 1 + 1 ) == 2;
  • console.log( answer ); // true

  • answer = ( 1 - 1 ) == 100;
  • console.log( answer ); // false

Ngoài ra, số 0 còn được hiểu ngầm là false và số 1 là true.

  • let bool = 0;
  • if( bool ) {
    console.log( "true" );
    } else {
    console.log( "false" );
    }
  • // "false"

null

Null là một kiểu dữ liệu đặc biệt, chỉ bao gồm một giá trị là null, được hiểu là không có giá trị.

  • let my_name = null;
  • console.log( my_name ); // null

undefined

Undefined củng là một kiểu dữ liệu đặc biệt trong JavaScript, chỉ bao gồm một giá trị undefined.

  • let my_name = undefined;
  • console.log( my_name ); // undefined

number

Number là kiểu dữ liệu dạng số (tương tự trong toán học). Number trong JavaScript không có cú pháp gì đặc biệt. Bạn chỉ cần viết số ra, có hai loại số là: int ( số nguyên ) và float ( số thực ).

  • let x = 1, // số nguyên dương
    x = 1.11, // số thực dương
    x = -1, // số nguyên âm
    x = -1.11, // số thực âm

string

String là kiểu dữ liệu dùng để biểu diễn chữ, văn bản, đoạn văn bản,...
Có ba cách để biểu diễn string trong JavaScript.

Dùng dấu nháy đơn '
Dùng dấu nháy đôi "
Dùng dấu backtick `
  • let my_name = "John";
  • my_name = 'Anna';
  • my_name = `Anna`;

Đặc biệt, đối với dấu `, bạn có thể sử dụng biến, hằng hoặc thậm chí viết một biểu thức trong đó, với cú pháp ${...}, ví dụ:

  • // Sử dụng biến
    let my_name = "John";
  • console.log( `My name is ${ my_name }` ); // "My name is John"

  • // Sử dụng với biểu thức
    console.log( `1 + 1 = ${ 1 + 1 }` ); // "1 + 1 = 2"

object

Object là kiểu dữ liệu tham chiếu, là một tập hợp gồm các cặp key - value ( khóa - giá trị ), được ngăn cách bởi dấu ,.
Trong đó, key có thể là string, value có thể là bất kì kiểu dữ liệu nào.

  • let my_self = {
    name
    "John"
    age
    100
    };

  • console.log( my_self.name ); // "John"
    console.log( my_self.age ); // 100
