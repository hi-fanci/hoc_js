jQuery

jQuery là một thư viện JavaScript phổ biến, được thiết kế để đơn giản hóa việc lập trình JavaScript trên các trang web.

Nó giúp lập trình viên thao tác với HTML, CSS, sự kiện, AJAX, và DOM dễ dàng hơn thông qua một cú pháp ngắn gọn và dễ sử dụng.

  • $(window).on( 'load', function() {
    • $('button').on( 'click', function() {
      • console.log( 'button clicked' );
      });
    });
  • $( ) đồng nghĩa với một hàm jQuery().
  • $(window).on( 'load', function() {...}) để đảm bảo mã chỉ chạy sau khi toàn bộ trang web đã được tải xong.
  • $('button').on( 'click', function() {...}) để xử lí hành động khi người dùng nhấn vào nút.

Thêm jQuery vào site

Để sử dụng jQuery bạn phải thêm thư viện vào web của mình, có 2 cái là .

  • Google CDN
    • <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
  • Tải về cục bộ

    Mình có thể tải jQuery ở đây.

    • <script src="./[...path]/jquery-3.6.0.min.js"></script>

    đảm bảo đường dẫn [...path] là chính xác.

Bộ chọn ( selector )

  • Chọn theo html tag
    • $('tag-name')
      // $('html'), $('body'), $('p')
  • Chọn theo id
    • $('#id')
      // $('#wraper'), $('#input'), $('#btn')
  • Chọn theo class
    • $('.class')
      // $('.toggler'), $('.confirm'), $('.submit')
  • Chọn nhiều
    • $('tag, #id, .class')
      // $('html, #wraper, .toggler')
  • Bộ chọn trong jQuery củng tương tự như bộ chọn trong css.
  • Phần tử được chọn luôn ở kiểu string.

Thao tác với HTML

Có rất nhiều hàm dựng sẳn trong jQuery, mình chỉ bày các hàm cơ bản thường gặp.

append()

Hàm này dùng để thêm nội dung mới vào cuối.

  • p
    • span This is text. /span
    /p
  • $('p').append('This is new text');
  • p
    • span This is text. This is new text /span
    /p
  • .append( )
    • $('p').append('This is new text'');

    hàm này giữ nguyên nội dung của thẻ và thêm nội dung mới vào ở cuối.

  • Thêm, thay đổi nội dung của phần tử.
    • // Sử dụng hàm html()
      $('p').html('<span>This is new text</span>'');

    • // Sử dụng hàm text()
      $('p').text('This is new text'');
    • Cả 2 hàm này đều dùng để thay đổi toàn bộ nội dung của thẻ và thay bằng nội dung mới.
    • Hàm text() chỉ có thể thêm mới dưới kiểu chuỗi, hàm html() thì có thể thêm các tag tuỳ ý.
  • Lấy nội dung của phần tử.
    • <p><span>This is text</span></p>
    • // Sử dụng hàm html()
      $('p').html();
      // <span>This is text</span>

    • // Sử dụng hàm text()
      $('p').text();
      // 'This is text'
  • Thay đổi thuộc tính của phần tử.
    • $('p').attr('data-name', 'This is new value');

    hàm này dùng để thay đổi giá trị của các thuộc tính trên thẻ, như src, data-, href, ... .
